Customer Demographics Analysis
📌 Project Overview
This project is part of my internship work.
It focuses on analyzing customer demographic data to understand customer behavior and patterns.
🎯 Objective
Analyze customer age, gender, location, and other demographics
Understand customer distribution
Extract insights for business decisions
🛠️ Tools Used
Google Colab
Python 🐍
Pandas
NumPy
Matplotlib / Seaborn
📂 Project Workflow
Dataset loaded in Google Colab
Data cleaning and preprocessing done
Exploratory Data Analysis (EDA) performed
Visualizations created for better insights
📊 Key Insights
Customer age group distribution analyzed
Gender-based customer split identified
Location-wise customer pattern studied
Business insights derived from demographics

Run all cells step by step
